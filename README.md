# GBX_Emulator
ゲームボーイのエミュレーターです  
音は未実装です  
動作しないソフトは多分あります  
<br>
# 実行可能ファイルについて
著作権の関係上実行可能ファイルは256バイトのbootromのバイナリファイルをどこかしらで手に入れて  
"bootrom/bootrom_256byte.bin" となるようにしてから自分でビルドしてください  
ビルドしてできた実行可能ファイルをExecutableディレクトリに入れると実行できます  
<br>
